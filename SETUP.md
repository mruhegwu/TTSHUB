# Installation and Setup Instructions for TTSHUB

## Prerequisites
- Ensure you have the latest version of Node.js installed.
- Install Python 3.x if not already available.

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/mruhegwu/TTSHUB.git
   cd TTSHUB
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Set up the environment variables. You can copy the example:
   ```bash
   cp .env.example .env
   ```
4. Modify the `.env` file according to your configuration.

## Running the Project
- To start the application, run:
  ```bash
  npm start
  ```
