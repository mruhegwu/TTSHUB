# API Documentation

## User Endpoints

### 1. Get All Users
- **Endpoint**: `/api/users`
- **Method**: GET
- **Description**: Retrieves a list of all registered users.

### 2. Get User by ID
- **Endpoint**: `/api/users/{id}`
- **Method**: GET
- **Description**: Retrieves a single user by their ID.

### 3. Create User
- **Endpoint**: `/api/users`
- **Method**: POST
- **Description**: Creates a new user.
